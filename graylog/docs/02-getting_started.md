# Getting Started 入门
为了便于您首次查看Graylog，我们提供了一个虚拟设备，可以使用VMware，VirtualBox等运行。在部署虚拟设备后，本节中的页面将帮助您介绍Graylog。

在您使用Graylog后感觉舒适后，您应该重新安装，以满足您的独特需求，例如尺寸和安全性。虚拟设备仅用于展示Graylog的功能，不应该在生产中运行。

如何您需要协助规划和构建您的日志环境，我们可以提供专业支持。
